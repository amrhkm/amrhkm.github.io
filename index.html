2022 Kaggle Machine Learning & Data Science Survey Dashboard

In this project, I visualized the data extracted from the survey answer. Using Power Query, a few changes to the dataset was made, such as changing some of the columns names to more suitable name, removing trailing spaces and shortening some of the responses (e.g., "Data Analyst (Business, Marketing, Financial, Quantitative, etc)" was changed to "data analyst"). DAX was also used to extract responses for the questions with sub-questions throughout multiple columns (e.g., Q6_1, Q6_2, Q6_3).

The target audience for this project is for someone like me who is pursuing a data profession from a different background to help them identify which technologies or studying platforms working data professionals use. Technologies requirements for a data profession can be overwhelming, having insights into what technologies other professionals use for their work will help us better decide on which tools to invest our time.

Winning Space Race With data science: SpaceX Falcon 9 Landing Analysis

This project analyzed past SpaceX launches that utilized the Falcon 9 rocket to understand the different attributes of the launches and their relationship with the success of Falcon 9â€™s first stage to land. The data for this project was collected from SpaceX API and scrapped from its Wikipedia launch page using Beautiful Soup in Python.

The data wrangling process was done in Python, where null values were checked, and landing outcome labels were extracted for each launch alongside exploratory data analysis (EDA) using Matplotlib and Seaborn. EDA process also used SQL (SQLAlchemy) to extract data from IBM DB2. Python was also used to build an interactive dash app that visualized the landing success rate across the launch sites and payload mass for various booster versions. Finally, a predictive model was built and tuned using Scikit-Learn to predict the landing outcome based on the launch attributes.

Skills: Python, Rest API, Webscrapping, EDA,

Poultry Pose Estimation by Detecting Keypoints with Deep Convolutional Network

With the advancement of artificial intelligence and specifically in the branch of the deep convolutional network, this research proposed an automated system that could detect the poultry's body keypoints using a deep convolutional network. Obtaining the body keypoints of poultry is a crucial step in developing an automated system that could detect the poultry's health state using its posture or mobility. However, this research focuses on posture estimation rather than mobility due to the type of data that could be gathered.

The data was collected from across the web from various sources, labelled using DeepLabCut GUI, and then trained using DeepLabCut in Python on Google Colab. This project focuses on comparing the accuracy of the proposed models to other models with different hyperparameters and backbones.

Skills: Convolutional Neural Network, Data Labelling, DeepLabCut, Python, Parameters Tuning
